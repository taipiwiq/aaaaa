# 🧠 教育機関向けクイズ管理・解答アプリ（開発中）

このアプリケーションは、学校や学習塾などの教育現場向けに設計された、クイズの作成・実施・成績管理を一元化するWebアプリです。  
Flask + PostgreSQL + Bootstrap で構築しており、ユーザー認証、ロール制御、作問・解答・履歴表示などの基本機能を備えています。
